# Перечень сокращений

| Сокращение | Расшифровка |
|------------|-------------|
| АС | автоматизированная система |
| БД | база данных |
| ВС | вычислительная система |
| DFD | диаграмма потоков данных (Data Flow Diagram) |
| ER | модель «сущность–связь» (Entity–Relationship) |
| ИС | информационная система |
| IDEF0 | Integration DEFinition for Function Modeling |
| IDEF1X | Integration DEFinition for Information Modeling |
| НСИ | нормативно-справочная информация |
| ООП | объектно-ориентированное программирование |
| ORM | объектно-реляционное отображение (Object–Relational Mapping) |
| ПИС | проектирование информационных систем |
| СППР | система поддержки принятия решений |
| СУБД | система управления базами данных |
| ТЗ | техническое задание |
| CRUD | создание, чтение, обновление, удаление (Create, Read, Update, Delete) |
| API | программный интерфейс приложения (Application Programming Interface) |
| CI/CD | непрерывная интеграция и доставка (Continuous Integration / Continuous Delivery) |
