# Приложение З. Образец результатного отчёта

Отчёт формируется в формате **XLSX** и сохраняется в `storage/app/public/reports/`.

## Структура файла

| Столбец | Содержание |
|---------|------------|
| № | Порядковый номер строки |
| Дата продажи / Дата покупки | `created_at` сделки |
| Автомобиль | mark + model |
| VIN | `cars.vin_code` |
| Сумма | `sails.price` (форматированная) |
| Клиент | имя и id |
| Менеджер | имя пользователя и id |

Последняя строка: **Итого** — количество сделок и суммарная выручка.

## Получение образца

1. Войдите в `/admin` под администратором.
2. Раздел «Отчёты» → создать отчёт типа «Продажа» за период с завершёнными сделками.
3. Дождитесь статуса «Завершён» (обработчик очереди: `make queue` или Horizon).
4. Скачайте файл по ссылке в записи отчёта.
5. Вставьте скриншот или приложите файл `report-sample.xlsx` к работе.
